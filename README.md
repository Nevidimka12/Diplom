# Многоуровневые акустические представления с причинно-временной структурой для непрерывного распознавания психофизиологических состояний человека

Выпускная квалификационная работа Абдулкадирова Тимура Назировича

## Описание

Работа посвящена разработке метода автоматического распознавания эмоциональных и личностных характеристик человека по речевому сигналу. 
Предлагается единая аудио-ориентированная модель, которая в многозадачной постановке одновременно решает задачи распознавания эмоций, оценки личностных черт и связанных поведенческих состояний.
